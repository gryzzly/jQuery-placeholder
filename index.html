jQuery-based shim to simulate placeholder attribute support in browsers that don't natively support it

Input with this script applied (check in IE)
Input without the script applied

Usage

Download the script, call it somewhere after you call jQuery and apply it to elements that have placeholder attribute (perhaps being specific is a good aidea, because of performance):

<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.4.3/jquery.min.js"></script>
<script src="path/to/jquery.placeholder.js"></script>
<script>
/* 
you'd need to put this in $(document).ready(function(){ //here }); wrapper,
if you aren't placing your scripts before closing </body> tag 
*/
$('input[placeholder], textarea[placeholder]').placeholder();
</script>
